Python basico (venv-first).

Instalar Python e venv:
```bash
yes | apt-get install python3
yes | apt-get install python3-pip
yes | apt-get install python3-venv
yes | apt-get install python-is-python3
```

Notas:
- Instale bibliotecas no venv (ver `31-python-venv-pytorch.md`).
- Evite `pip install` no sistema base.

Referencia:
- https://docs.python.org/3/library/venv.html
