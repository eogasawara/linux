Pacotes basicos e atualizacoes.

Ferramentas basicas:
```bash
yes | apt install vim
yes | apt install ssh
yes | apt install net-tools
yes | apt install htop
yes | apt install btop
```

Ciclo de atualizacao:
```bash
yes | apt update
yes | apt upgrade
yes | apt autoremove
```

Checagens rapidas:
```bash
uname -a
lsb_release -a
```
